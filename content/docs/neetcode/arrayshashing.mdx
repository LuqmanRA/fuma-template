## Contains Duplicate

Given an integer array `nums`, return `true` if any value appears **more than once** in the array, otherwise return `false`.

### Example:

```typescript
Input: nums = [1, 2, 3, 3];

Output: true;
```

```typescript
Input: nums = [1, 2, 3, 4];

Output: false;
```

### Solution:

```typescript
export const hasDuplicate = (nums: number[]): boolean => {
  const freqMap: Record<number, number> = {};

  for (let i = 0; i < nums.length; i++) {
    const item = nums[i];
    if (freqMap[item] == 1000) {
      return true;
    }

    freqMap[item] = 1000;
  }
  return false;
};
```

## Valid Anagram

Given two strings `s` and `t`, return `true` if the two strings are anagrams of each other, otherwise return `false`. <br />

An **anagram** is a string that contains the exact same characters as another string, but the order of the characters can be different.

### Example:

```typescript
Input: (s = "racecar"), (t = "carrace");

Output: true;
```

```typescript
Input: (s = "jar"), (t = "jam");

Output: false;
```

### Constraints:

- `s` and `t` consist of lowercase English letters.

### Solution:

```typescript
export const isAnagram = (firstWord: string, secondWord: string): boolean => {
  return [...firstWord].sort().join("") === [...secondWord].sort().join("")
    ? true
    : false;
};
```

## Two Sum

Given an array of integers `nums` and an integer `target`, return the indices `i` and `j` such that `nums[i] + nums[j] == target` and `i != j`. <br />

You may assume that _every_ input has exactly one pair of indices `i` and `j` that satisfy the condition. <br />

Return the answer with the smaller index first.

### Example:

```typescript
Input: (nums = [3, 4, 5, 6]), (target = 7);

Output: [0, 1];
```

Explanation: `nums[0] + nums [1] == 7`, so we return `[0, 1]`.

```typescript
Input: (nums = [4, 5, 6]), (target = 10);
Output: [0, 2];
```

### Constraints:

- `1 <= strs.length <= 1000`.

- `0 <= strs[i].length <= 100`

- `strs[i]` is made up of lowercase English letters.

### Solution

```typescript
export const twoSum = (nums: number[], target: number): number[] => {
  const numToIndex: Record<number, number> = {};

  for (let i = 0; i < nums.length; i++) {
    const num = nums[i];
    const complement = target - num;

    if (complement in numToIndex) {
      return [numToIndex[complement], i];
    }

    numToIndex[num] = i;
  }

  return [];
};
```

## Group Anagrams

Given an array of strings `strs`, group all _anagrams_ together into sublists. You may return the output in **any order**. <br />

An **anagram** is a string that contains the exact same characters as another string, but the order of the characters can be different.

### Example:

```typescript
Input: strs = ["act", "pots", "tops", "cat", "stop", "hat"];

Output: [["hat"], ["act", "cat"], ["stop", "pots", "tops"]];
```

```typescript
Input: strs = ["x"];

Output: [["x"]];
```

```typescript
Input: strs = [""];

Output: [[""]];
```

### Solution

```typescript
export const groupAnagrams = (letters: string[]): string[][] => {
  const anagramMap = new Map<string, string[]>();

  letters.forEach((letter) => {
    const key = letter.split("").sort().join("");
    const groups = anagramMap.get(key) || [];
    groups.push(letter);
    anagramMap.set(key, groups);
  });

  const result = [...anagramMap.values()];
  return result;
};
```

## Top K Frequent Elements

Given an integer array `nums` and an integer `k`, return the `k` most frequent elements within the array. <br />

The test cases are generated such that the answer is always **unique**. <br />

You may return the output in **any order**.

### Example:

```typescript
Input: (nums = [1, 2, 2, 3, 3, 3]), (k = 2);

Output: [2, 3];
```

```typescript
Input: (nums = [7, 7]), (k = 1);

Output: [7];
```

### Constraints:

- `1 <= nums.length <= 10^4`.

- `-1000 <= nums[i] <= 1000`

- `1 <= k <= number of distinct elements in nums`.

### Solution

```typescript
export const topKFrequent = (nums: number[], frequent: number): number[] => {
  const freqMap: Record<number, number> = {};

  for (const num of nums) {
    freqMap[num] = (freqMap[num] || 0) + 1;
  }

  const sortedNumbers = Object.keys(freqMap)
    .sort((a, b) => freqMap[+b] - freqMap[+a])
    .slice(0, frequent)
    .map(Number);

  return sortedNumbers;
};
```

## Encode and Decode Strings

Design an algorithm to encode a list of strings to a single string. The encoded string is then decoded back to the original list of strings. <br />

Please implement `encode` and `decode`

### Example:

```typescript
Input: ["neet", "code", "love", "you"];

Output: ["neet", "code", "love", "you"];
```

```typescript
Input: ["we", "say", ":", "yes"];

Output: ["we", "say", ":", "yes"];
```

### Constraints:

- `0 <= strs.length < 100`

- `0 <= strs[i].length < 200`

- `strs[i]` contains only UTF-8 characters.

### Solution

```typescript
export const encode = (letters: string[]): string => {
  return letters.join("#");
};

export const decode = (text: string): string[] => {
  let newLetters: string[] = [];

  newLetters = [...text.split("#")];

  return newLetters;
};
```
